SASS

SASS is a CSS preprocessor that allows you to modularize your CSS files. This means edits are easier to make and the overall code is a lot cleaner. I have over a year of experience using SASS. SASS also allows for variables. This is important for all major companies. The reason it is important is that with the use of variables we can change a variable color and everything that calls that variable will change in color. It makes it very nice when we need to change out primary colors used across the site.
